## Key Observations

- **ADK 2.0 deterministic workflows + no visual tooling** = the LangGraph vs ADK debate will be won by whichever has better developer inspection tooling, not better architecture. ADK 2.0 has the right model; the ecosystem needs to see it.
- **Cloud-hosted MCP is the next table stakes.** Azure shipped it. Google has API Registry as a managed tool registry but not a running MCP endpoint. The missing piece is making tools _callable_ as managed cloud services without running your own server.
- **OpenAI SDK v0.13.1 is now multi-cloud.** Provider-agnostic across 100+ LLMs + Sessions + Realtime voice = a complete agent SDK that runs everywhere including on GCP. This is a distribution threat: developers build on OpenAI SDK and deploy to any cloud. ADK needs a stronger "GCP-native advantages" story.
- **Security is M&A territory.** OpenAI bought Promptfoo. 1Password ships agent credential vaults. HiddenLayer ships agentic runtime security. GCP's Model Armor + Agent Identity is fragmented. Unified security surface is a product gap and competitive moat opportunity.
