## 4. Papers Worth Reading (Top 4)

- [Meta-Cognitive Memory Policy Optimization (MMPO)](https://arxiv.org/abs/2605.30159) by Liu, Hao et al. — Belief Entropy scoring + MMPO loss achieves **97.1% task performance at 1.75M-token contexts**. Current memory compaction is uncertainty-blind; MMPO prevents lossy summarization of ambiguous states. Directly applicable to GEAP Memory Bank compaction design.

- [Locally Coherent, Globally Incoherent](https://arxiv.org/abs/2605.30335) by Kotawala — **33–94% of multi-agent configurations** violate global probability axioms while appearing locally consistent. Retrieval augmentation provides limited mitigation. **No platform addresses this today. ADK first-mover opportunity.**

- [SkillsInjector: Dynamic Skill Context Construction](https://arxiv.org/abs/2605.29794) by Li, Liu, Gao et al. — Two-stage dynamic skill injection (adaptive context planning + custom rendering) delivers **3.9–7.3pp improvement** over static baselines. ADK Tool Registry is static admin-managed. Dynamic injection is the next ADK capability tier.

- [Long Live the Librarian](https://arxiv.org/abs/2605.27787) by Cho, Choi et al. — Output tokens consume 30–1000x more GPU energy than input. Persistent Librarian sub-agent eliminates redundant repo exploration. **25% GPU energy reduction** at same performance. Maps directly to Agent Engine token cost economics for enterprise SWE workloads.

## 5. Competitive Landscape

**AWS ahead of GCP:**
- AgentCore Payments: managed wallet + session spending limits (GCP has AP2 spec, not execution)
- AgentCore WorkSpaces: managed desktop automation via Firecracker microVMs (GCP has no equivalent)
- AgentCore Optimization: closed observe→eval→optimize→deploy loop (GCP pieces exist but disconnected)
- Per-session Firecracker microVM isolation (GCP Agent Engine runs on shared Cloud Run)

**Azure ahead of GCP:**
- Browser Automation Tool (GA Playwright sessions) — GCP has no managed equivalent
- Computer Use Tool (Preview) — GCP requires developers to wire Gemini 2.5 CU manually
- SRE Agent (GA, 35K+ incidents/month automated) — no GCP customer-facing equivalent
- Cosmos DB BYO thread storage — rough parity with GEAP Sessions

**GCP ahead of competitors:**
- Gemini 3.1 Pro: 77.1% ARC-AGI-2 (2x Gemini 3 Pro) — highest reasoning score across platforms
- A2A protocol v1.0: in production at 150 orgs; AWS/Azure have no open standard equivalent
- ADK 5-language support (Python/TS/Go/Java/Kotlin) — OpenAI Agents SDK: Python+TS only
- Google Search AI Agents + Gemini Spark: **only cloud running own agent platform at consumer scale (billions of users)**
- AP2/Mastercard: better-designed open payment protocol than AWS's x402
