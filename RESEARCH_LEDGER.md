# Research Ledger — AI Agent Intelligence Tracker

**Source of truth** for all AI agent products, papers, and platform updates discovered by Scout and Scholar.

**Updated by:** Scout (products), Scholar (papers), Analyst (status changes)
**Reviewed by:** Yujun (weekly — mark items as Actioned/Archived)

---

## How to Use

- **Scout** appends new rows after each daily scan
- **Scholar** appends new rows after each paper scan
- **Analyst** references this for daily briefings, updates Status column
- **Yujun** reviews weekly, marks items Actioned/Archived, adds Action Taken notes

## Status Legend

| Status | Meaning |
|--------|---------|
| 🆕 New | Just discovered, not yet reviewed |
| 👀 Tracked | Being monitored for updates |
| ✅ Actioned | Shared with team / built POC / integrated |
| 📦 Archived | No longer relevant |
